Materi Pertemuan 4: Pengenalan PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman sisi server yang digunakan untuk membuat halaman web dinamis. PHP dapat dijalankan di berbagai server seperti Apache dan kompatibel dengan banyak database seperti MySQL.

PHP dapat disisipkan dalam HTML menggunakan tag <?php ... ?>. Contoh sederhana untuk mencetak teks adalah <?php echo "Hello, World!"; ?>. File PHP harus disimpan dengan ekstensi .php.

PHP mendukung variabel yang diawali dengan simbol $. Misalnya, $nama = "John"; akan menyimpan teks "John" dalam variabel $nama. PHP juga memiliki struktur kontrol seperti if-else dan perulangan seperti for dan while.

PHP sering digunakan untuk mengakses database. Dengan fungsi mysqli_connect(), PHP bisa terhubung ke MySQL dan melakukan operasi seperti insert, update, dan delete data, sehingga cocok untuk membuat aplikasi berbasis web.
