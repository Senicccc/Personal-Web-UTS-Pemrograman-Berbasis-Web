Materi Pertemuan 3: Javascript

p>JavaScript adalah bahasa pemrograman yang digunakan untuk membuat halaman web lebih interaktif. JavaScript dapat mengontrol elemen HTML, menangani event, dan melakukan komunikasi dengan server tanpa perlu memuat ulang halaman.

JavaScript dapat ditulis langsung di dalam tag <script> atau di file eksternal dengan ekstensi .js. Contoh sederhana penggunaan JavaScript adalah alert("Hello, World!"); yang akan menampilkan pop-up pesan di browser.

JavaScript menggunakan variabel yang dapat dideklarasikan dengan var, let, atau const. Contohnya, let nama = "John"; akan menyimpan teks "John" dalam variabel nama. JavaScript juga mendukung struktur kontrol seperti if-else dan perulangan seperti for dan while.

JavaScript sering digunakan untuk menangani event seperti klik tombol. Misalnya, dengan document.getElementById("btn").addEventListener("click", function() { alert("Tombol diklik!"); });, kita bisa membuat tombol yang menampilkan peringatan saat diklik.
