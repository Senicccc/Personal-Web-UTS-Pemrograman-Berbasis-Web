Materi Pertemuan 6: Perulangan

Perulangan digunakan untuk menjalankan suatu blok kode secara berulang selama kondisi tertentu terpenuhi. Perulangan dapat membantu mengurangi kode yang berulang dan meningkatkan efisiensi program.

Salah satu jenis perulangan adalah for, yang digunakan jika jumlah iterasi sudah diketahui. Contohnya: for (let i = 1; i <= 5; i++) { console.log(i); } akan mencetak angka 1 hingga 5.

Perulangan while digunakan jika jumlah iterasi belum pasti. Contohnya: let i = 1; while (i <= 5) { console.log(i); i++; } Kode ini juga akan mencetak angka 1 hingga 5, tetapi lebih fleksibel dalam penggunaannya.

Ada juga perulangan do-while, yang mirip dengan while tetapi pasti dijalankan minimal sekali: let i = 1; do { console.log(i); i++; } while (i <= 5); Perulangan ini akan mencetak angka 1 hingga 5, meskipun kondisi awalnya tidak memenuhi syarat.

Selain perulangan, switch digunakan untuk memilih satu dari beberapa kemungkinan berdasarkan nilai variabel. Contohnya: let hari = 3; switch (hari) { case 1: console.log("Senin"); break; case 2: console.log("Selasa"); break; case 3: console.log("Rabu"); break; default: console.log("Hari tidak valid"); } Jika variabel hari bernilai 3, maka akan mencetak "Rabu".
