Materi: Array

Array dalam PHP digunakan untuk menyimpan banyak nilai dalam satu variabel. Array dapat menyimpan berbagai tipe data, seperti string, angka, atau bahkan array lainnya.

Ada tiga jenis array dalam PHP: Array indeks numerik, Array asosiatif, dan Array multidimensi.

Contoh array indeks numerik: $buah = array("Apel", "Mangga", "Jeruk"); echo $buah[1]; // Output: Mangga

Contoh array asosiatif, yang menggunakan kunci untuk menyimpan nilai: $umur = array("Ali" => 25, "Budi" => 30); echo $umur["Budi"]; // Output: 30

Contoh array multidimensi: $mahasiswa = array( array("Ali", 20, "Informatika"), array("Budi", 22, "Ekonomi") ); echo $mahasiswa[0][0]; // Output: Ali

Untuk menampilkan semua isi array, bisa menggunakan perulangan foreach: foreach ($buah as $b) { echo $b . " "; } Kode ini akan mencetak: Apel Mangga Jeruk.
